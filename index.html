HTML

  • The head elements contains information about the webpage
  • !DOCTYPE tells the browser the document type it should expect when its opened
  • Meta contains info about the page that is used by the browser
  • The body element represents the visible content shown to the user
  • Tag refers to what is inside the angle brackets
  • Title tag is the name displayed on browser tab
  • Elements refers to the opening tag to closing tag
  • Child elements are elements nested inside other elements
  • If three elements are nested inside the body element, they are known as child elements and the body element is refered to the parent element
  • div tag creates a division in the page
  • span tag creates an inline division
  • h1-h6 elements represent the level of heading a given text block represents
  • p element represents a paragraph or block of text
  • ul, ol, li elements are unordered lists, ordered lists, and list items respectively
  • img element contains the info about the image
  • src attribute within img element defines the location of the image
  • alt attribute within img element contains a text string that describes the appearance and functionality of the image
  • a element (anchor) creates links to the same or other webpage(s)
  • href attribute points to the URL for the link
  • footer element appears at the bottom of the page
  • section elements creates sections in the page

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • CSS box model shown below
  • Best practice to use external CSS style sheet instead of internal style sheet and/or inline CSS
  • Separation of concerns refers to separating code into different files based on use
  • CSS rule shown below
  • Using * as the selector means we want all elements on the page to have the style properties
  • Mutiple selectors can be combined into a single CSS rule as long as they are separated by commas
  • Make sure to link the CSS file in the HTML code via the link element
  • Use a class selector by doing .classname
  • Assigning colors in CSS can be done by either typing the color or using its hexadecimal code (preferred method)

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git clone: clones repo into local branch
  • git add -A: adds code to current working branch
  • git commit -m "": commits a message that describes the changes made to the code
  • git pull origin main: checks if local branch is in sync with base branch in GitHub
  • git push origin branchname: push changes to GitHub branch
